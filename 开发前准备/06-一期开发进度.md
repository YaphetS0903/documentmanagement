- 已实现分类树的新建下级、编辑、移动、删除。

- 已实现公告管理，支持公告列表、草稿、发布、撤销、删除、范围和附件。

- 已实现后端 REST API，覆盖登录、组织、文档、版本、权限、搜索、消息、收藏、分享、发布、订阅、提醒、附件、关联文件、分类、属性、评论、评分、公告、个人网盘、开放 API、回收站、审计。

- 已实现权限继承范围细分语义和管理员代用户权限预览。
- 已实现文件上传策略，支持扩展名白名单和单文件大小限制。
- 已实现文件业务状态维护、搜索高亮、搜索高级筛选和排序。
- 已实现文档库分页控件和工作台近 7 天文件增长趋势。
- 已实现消息详情、已读、关联文件跳转。
- 已实现开放 API 凭证、AccessKey/Secret 鉴权、一次性登录票据、调用日志、耗时记录、基础限流和 OpenAPI JSON。
- 已实现审计日志 CSV 导出。
- 已实现个人网盘基础空间、空间概况、私密性隔离和复制到企业文档库。
- 已实现管理员模块按角色展示，普通用户不再显示用户、组织、公告管理、开放 API、审计日志入口。
- 已通过 390px 移动视口响应式验证，移动导航与内容宽度正常。
- 已实现旧系统核心目录初始化。
- 已通过生产构建。
- 已通过扩展 API 冒烟测试。
- 已通过浏览器验证：登录页、工作台、文档库、知识地图、文本预览均可正常渲染。

## 仍需继续完善

- 在线编辑：需要接 OnlyOffice 或 WPS WebOffice。
- 生产数据库：当前是 JSON 文件存储，后续需迁移到 PostgreSQL/MySQL。
- 生产文件存储：当前是本地目录，后续需接 MinIO/NAS。
- 更强全文检索：当前是内置搜索，后续需接 OpenSearch/Elasticsearch。
- PDF 正文抽取和扫描件 OCR。
- 企业微信免登、组织同步、消息推送。
- Swagger UI、网关级限流、OAuth2/SSO API 等开放平台生产化能力。
- 自动化测试继续扩展更多权限边界和文件操作。
