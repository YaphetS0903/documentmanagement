# 文档管理平台一期开发版

这是基于前期需求文档启动的 B/S 文档管理平台一期开发版本。

## 已实现能力

- 账号登录、退出、当前用户信息。
- 登录安全支持验证码、密码策略、账号禁用、失败审计和连续错误临时锁定。
- 用户、部门、角色基础管理，支持部门/角色树节点新增、编辑、删除和挂接上级。
- 企业公共文档库目录树。
- 旧系统业务目录初始化：董事会、TES、公司管理、知识管理、公司大事记。
- 文件夹创建、重命名、移动、复制、删除。
- 文件上传、批量上传。
- 上传策略配置，支持扩展名白名单和单文件大小限制。
- 文件下载、文件夹打包下载、批量下载。
- 文件版本列表、上传新版本、历史版本预览、版本回滚。
- 文件锁定、解锁。
- 文件/文件夹权限规则管理。
- 权限规则支持允许/拒绝、优先级、细分继承范围和文件名/路径/扩展名/业务状态条件。
- 管理员可预览指定用户对文件/文件夹的最终权限。
- 文件分享、发布，并向接收人发送消息。
- 分享/发布记录管理，可撤销。
- 文件/文件夹订阅，更新或删除时向订阅者发送消息。
- 订阅记录管理，可取消。
- 文件闹钟提醒，消息中心/工作台会检查到期提醒。
- 文件闹钟记录管理，可编辑周期、提醒方式、结束时间，可取消。
- 公告管理，支持草稿、发布、撤销、范围和附件。
- 文件附件上传、列表、下载、删除。
- 文件关联关系创建、列表、预览关联文件、删除。
- 回收站、恢复、彻底删除。
- 分类树、文件分类、标签和扩展属性，支持分类树维护、属性定义编辑、必填和枚举选项。
- 文件详情弹窗，支持基础信息、业务状态、标签、分类、扩展属性、评分和评论。
- 文件评论、评分。
- 文件名和正文搜索，支持类型、创建人、路径、更新时间、排序、分页和结果高亮。
- 标签、分类、扩展属性参与搜索。
- txt/md/json/xml/html 等文本预览。
- docx/xlsx/pptx 文本抽取、检索和文本预览兜底。
- PDF、图片浏览器预览。
- 收藏、消息中心、消息详情、关联文件跳转、工作台统计、文件增长趋势、最近操作。
- 个人中心、修改密码。
- 管理员模块按角色显示，普通用户只看到可访问的业务入口。
- 个人网盘基础空间，支持空间概况、私密性隔离、复用文档库文件操作并复制到企业文档库。
- 审计日志查询和 CSV 导出。
- 开放 API 凭证管理、AccessKey/Secret 鉴权、一次性登录票据、调用日志、基础限流和 OpenAPI JSON。


## 本地运行

安装依赖：

```bash
npm install --cache .npm-cache
```

初始化数据：

```bash
npm run seed
```

开发模式：

```bash
npm run dev
```

生产构建：

```bash
npm run build
```

启动生产构建：

```bash
npm run start
```

访问地址：

```text
http://localhost:3000
```

## 当前技术说明

当前版本为了快速跑通一期主流程，使用：

- 前端：Vue 3、Vite、Element Plus、Lucide 图标。
- 后端：Node.js、Express。
- 数据：文件型 JSON 数据库。
- 文件：本地上传目录。

后续生产化建议替换为：

- PostgreSQL/MySQL。
- MinIO/NAS/对象存储。
- OpenSearch/Elasticsearch。
- OnlyOffice/WPS WebOffice。
